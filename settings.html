Welcome to Recess

Let's get your leave balances set up — about 2 minutes.

Your data is 100% private

Recess is a static web page — there is no server. Everything you enter stays on this device in your browser's local storage.

Your balances, service date, and usage history are never transmitted anywhere. The source code is open and auditable on GitHub.

Export your data to a JSON file at any time and import it on any device.

Leave Accrual

Used to calculate your accrual tier (4/6/8 hrs per pay period). Mid-year tier changes are handled automatically.

Starting Balances

Important: Enter the balance from your most recent leave statement and set the date to the first day of that pay period. These are hours carried into that period — not the post-accrual total. Using the wrong value will offset all future projections by one pay period of accrual.

Pay Schedule

The anchor date is any known pay period start date — all future dates are derived from it automatically.
